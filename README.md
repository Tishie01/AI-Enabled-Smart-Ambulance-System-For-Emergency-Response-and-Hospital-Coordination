# AI-Enabled-Smart-Ambulance-System-For-Emergency-Response-and-Hospital-Coordination

## Features

- **Ambulance Dashboard**: Paramedics can create sessions, fill patient and guardian details, and choose between automatic (IoT-based) or manual health data entry.
- **Real-time Health Monitoring**: Heart rate, body temperature, and blood oxygen levels are streamed live to the guardian.
- **Guardian Access**: Guardians receive an SMS with a session link and OTP. They enter their NIC to verify and join the session.
- **Live Chat**: Two-way chat between paramedic and guardian during transit.
- **Session Status Tracking**: Display "On the way" during active sessions and "Arriving to hospital" when stopping.
- **Automatic Summary SMS**: When the session ends, a health data summary is automatically sent to the guardian's contact number.
